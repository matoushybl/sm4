pub struct Duration { /* fields omitted */ }

A Duration type to represent a span of time.

This data type is only available on ARMv7-M

Correctness

This type is not appropriate for representing time spans in the order of, or larger than, seconds because it can hold a maximum of (1 << 31) "ticks" where each tick is the inverse of the CPU frequency, which usually is dozens of MHz.
